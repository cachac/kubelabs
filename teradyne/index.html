Plan del Curso: Kubernetes & CI/CD

Dos rutas de aprendizaje estructuradas de forma lógica y secuencial, basadas en la infraestructura nativa de la nube.

Ruta 1 3 lecciones (4 horas c/u)

🚀 Kubernetes

Dominio práctico del orquestador. Desde conceptos esenciales de despliegue hasta la automatización de escalabilidad y enrutamiento avanzado.

  • Despliegue de Pods y ReplicaSets deploy pods rs

    Aprende los principios atómicos del diseño de workloads, control y estrategias de despliegue seguro.

  • Autoscaling HPA Avanzado hpa advanced

    Configuración avanzada de escalado automático basado en demandas y límites reales de CPU/Memoria.

  • Ingress & Gateway API ingress gw api

    Gestión y control moderno del tráfico externo. Transición de Ingress tradicional al nuevo estándar Gateway API.

  • AIOps AI ops

    Operaciones optimizadas con Inteligencia Artificial para la configuración dinámica de cargas de trabajo.

Ruta 2 5 lecciones (4 horas c/u)

🛠️ CI/CD & GitOps

Automatización total del ciclo de entrega. Creación de flujos de integración eficientes y despliegue continuo impulsado por el estado de Git.

  • GitHub Actions PR / workflows

    Automatización integral de flujos de integración continua (CI) disparados por eventos de Pull Requests.

  • Kustomize KUSTUMIZE

    Personalización declarativa de plantillas y recursos según tus ambientes (dev, qa, prod) sin sobreescribir código base.

  • GitOps con Argo CD ARGO CD

    Sincronización automatizada entre tus repositorios y el clúster de Kubernetes, garantizando resiliencia y control.

  • Despliegue a Producción MAIN

    Implementación del flujo de producción continuo desde la integración en la rama principal.
